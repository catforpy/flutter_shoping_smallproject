# UI 组件架构设计

## 层次划分

### 一、核心能力层 (core/)
**职责**：提供最基础的 UI 能力，不包含业务逻辑

```
core/
└── core_ui/
    ├── lib/src/
    │   ├── extensions/        # Widget 扩展方法
    │   │   ├── widget_extensions.dart
    │   │   ├── list_extensions.dart
    │   │   └── string_extensions.dart
    │   ├── animations/        # 基础动画封装
    │   │   ├── fade_animation.dart
    │   │   ├── scale_animation.dart
    │   │   └── shake_animation.dart
    │   └── themes/            # 主题相关
    │       └── app_theme.dart
    └── pubspec.yaml
```

**特点**：
- ✅ 只有扩展方法和基础工具
- ✅ 不包含完整组件
- ✅ 所有项目都可以复用

## 组件封装示例

### 1. Widget 扩展方法 (ui_extensions)

```dart
// lib/src/extensions/widget_extensions.dart
extension WidgetExtensions on Widget {
  /// 链式调用：添加内边距
  Widget padding({double? all, double? horizontal, double? vertical}) {
    return Padding(
      padding: EdgeInsets.only(
        left: horizontal ?? all ?? 0,
        right: horizontal ?? all ?? 0,
        top: vertical ?? all ?? 0,
        bottom: vertical ?? all ?? 0,
      ),
      child: this,
    );
  }

  /// 链式调用：添加圆角
  Widget borderRadius({double? all}) {
    return ClipRRect(
      borderRadius: BorderRadius.circular(all ?? 0),
      child: this,
    );
  }

  /// 链式调用：添加点击手势
  Widget onTap(VoidCallback onTap) {
    return GestureDetector(
      onTap: onTap,
      child: this,
    );
  }
}
```

**使用方式**：
```dart
// 传统写法
Padding(
  padding: EdgeInsets.all(16),
  child: ClipRRect(
    borderRadius: BorderRadius.circular(8),
    child: GestureDetector(
      onTap: () {},
      child: Text('Hello'),
    ),
  ),
)

// 使用扩展方法
Text('Hello')
  .padding(all: 16)
  .borderRadius(all: 8)
  .onTap(() {})
```

### 2. 横向滑动标签组件 (ui_components)

```dart
// lib/src/components/tabs/horizontal_tabs.dart
class HorizontalTabs extends StatefulWidget {
  final List<TabItem> tabs;
  final int currentIndex;
  final ValueChanged<int> onTap;
  final int? pinnedTabIndex;  // 要吸附的标签索引

  const HorizontalTabs({
    required this.tabs,
    required this.currentIndex,
    required this.onTap,
    this.pinnedTabIndex,
  });
}

class _HorizontalTabsState extends State<HorizontalTabs> {
  bool _showPinnedTab = false;

  @override
  Widget build(BuildContext context) {
    return Row(
      children: [
        // 固定标签（条件显示）
        if (_showPinnedTab && widget.pinnedTabIndex != null)
          _buildPinnedTab(),

        // 可滑动标签区域
        Expanded(
          child: ListView.builder(
            scrollDirection: Axis.horizontal,
            itemCount: widget.tabs.length,
            itemBuilder: (context, index) {
              return TabWidget(
                tab: widget.tabs[index],
                isSelected: widget.currentIndex == index,
                onTap: () => widget.onTap(index),
              );
            },
          ),
        ),
      ],
    );
  }
}
```

**使用方式**：
```dart
HorizontalTabs(
  tabs: [
    TabItem(id: 0, title: '关注'),
    TabItem(id: 1, title: '推荐'),
    TabItem(id: 2, title: '热门'),
  ],
  currentIndex: _currentIndex,
  onTap: (index) => setState(() => _currentIndex = index),
  pinnedTabIndex: 1,  // 推荐 tab 吸附
)
```

## 对比 DudaFlutter/common 的结构

### DudaFlutter/common 结构
```
common/
├── widget_extensions.dart    # ✅ Widget 扩展方法
├── common_sticky/            # ✅ 吸顶组件
├── animations/               # ✅ 动画封装
└── utils/                    # ✅ 工具类
```

### 我们的架构
```
core/core_ui/                # 对应 common/ 的基础部分
packages/ui/                  # 对应 common/ 的组件部分
├── ui_extensions/           # → widget_extensions.dart
├── ui_components/           # → common_sticky/
└── ui_templates/            # 新增：模版系统
```

## 总结

| 问题 | 答案 |
|------|------|
| **组件放在哪一层？** | 放在 `packages/ui/` 下，分为 ui_extensions、ui_components、ui_templates |
| **是否需要扩展方法？** | 是！创建 `ui_extensions` 包，参考 `widget_extensions.dart` |
| **是否需要动画封装？** | 是！在 `ui_components` 中包含动画组件 |
| **是否需要 sticky 组件？** | 是！参考 `common_sticky` 实现 |
