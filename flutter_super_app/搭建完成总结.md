### 第二步：选择一个 Package 开始开发

**推荐顺序**（从下往上）：

```
1. core_base（基础类型）
   - 打开：packages/core/core_base/lib/core_base.dart
   - 定义基础类型

2. core_utils（工具函数）
   - 打开：packages/core/core_utils/lib/core_utils.dart
   - 编写工具方法

3. core_network（网络请求）
   - 打开：packages/core/core_network/lib/core_network.dart
   - 封装 HTTP 客户端
```

### 第三步：添加依赖关系

```
假设你在开发 core_network，需要使用 core_utils 的工具：

1. 打开 packages/core/core_network/pubspec.yaml
2. 在 dependencies 部分添加：

dependencies:
  core_utils:
    path: ../core_utils

3. 运行：melos bootstrap
```

## 📖 如何使用这个项目

### 场景 1：我要添加一个工具函数

```
1. 找到对应的 Package：core_utils
   路径：packages/core/core_utils/

2. 在 lib 文件夹创建 .dart 文件
   例如：lib/src/extensions/string_extension.dart

3. 编写代码

4. 在 core_utils.dart 中导出
   export 'src/extensions/string_extension.dart';
```

### 场景 2：我要添加一个新的业务模块

```
1. 创建 Package
   cd packages/modules
   flutter create --template=package module_video

2. 在 melos.yaml 中已经配置了 packages/**
   所以会自动管理

3. 运行 melos bootstrap
```

### 场景 3：我要让一个 Package 依赖另一个 Package

```
例如：module_shop 依赖 service_user

1. 打开 packages/modules/module_shop/pubspec.yaml

2. 在 dependencies 添加：
   dependencies:
     service_user:
       path: ../../services/service_user

3. 运行 melos bootstrap
```
