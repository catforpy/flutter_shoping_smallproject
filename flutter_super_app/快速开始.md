## 🎯 第一次开发：试试创建一个工具函数

### 目标：在 core_utils 中添加一个字符串工具

### 步骤：

**1. 打开 core_utils Package**

```bash
cd packages/core/core_utils
```

**2. 查看现有文件**

```bash
ls -la lib/
```

**3. 创建一个新的工具文件**

```bash
mkdir -p lib/src/extensions
touch lib/src/extensions/string_extension.dart
```

**4. 编辑文件**

```bash
# 用你喜欢的编辑器打开
open lib/src/extensions/string_extension.dart
```

**5. 添加代码**

```dart
/// String 扩展方法
extension StringExtension on String {
  /// 判断是否为空
  bool get isNullOrEmpty => this == null || this!.isEmpty;

  /// 判断是否不为空
  bool get isNotNullOrNotEmpty => !isNullOrEmpty;

  /// 验证是否是手机号
  bool get isPhoneNumber {
    if (isNullOrEmpty) return false;
    return RegExp(r'^1[3-9]\d{9}$').hasMatch(this!);
  }
}
```

**6. 在主导出文件中导出**

编辑 `lib/core_utils.dart`，添加：

```dart
export 'src/extensions/string_extension.dart';
```

**7. 测试**

```bash
cd ../..
melos run test
```

## 🎓 核心概念速记

``packages/    = 所有代码包
core/         = 基础层（底层）
services/    = 服务层（业务）
modules/     = 模块层（功能）
apps/        = 应用层（最终产品）

melos        = 管理所有包的工具
pubspec.yaml  = 包的配置文件（类似 package.json）
lib/         = 代码文件夹
```
