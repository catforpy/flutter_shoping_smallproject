# 主应用 (app_main)

Flutter 超级应用主应用 - 基础框架

## 当前功能

### 基础导航框架
- ✅ 底部导航栏（2个Tab）
- ✅ 顶部导航栏（动态标题）
- ✅ 首页页面
- ✅ 我的页面

## 运行方式

### 方法1：使用 Melos
```bash
# 在项目根目录
melos run start
```

### 方法2：直接运行
```bash
cd apps/app_main
flutter run
```

## 项目结构

```
apps/app_main/
├── lib/
│   ├── main.dart              # 应用入口
│   └── src/
│       ├── app.dart            # 主框架（底部导航栏）
│       └── pages/
│           ├── home_page.dart  # 首页
│           └── profile_page.dart # 我的页面
└── pubspec.yaml
```
