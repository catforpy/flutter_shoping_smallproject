## 3. 状态管理

### 3.1 Riverpod

```yaml
dependencies:
  flutter_riverpod: ^2.4.0
  riverpod_annotation: ^2.3.0

dev_dependencies:
  riverpod_generator: ^2.3.0
  riverpod_lint: ^2.3.0
  build_runner: ^2.4.0
```

**选择理由**

```
✅ 类型安全: 编译时检查 provider 依赖
✅ 无需 BuildContext: 不依赖 Widget 树
✅ 代码生成: 自动生成 provider，减少样板代码
✅ 易于测试: 不依赖 Flutter，可单独测试
✅ 性能优秀: 细粒度更新，避免不必要的重建
```

**使用示例**

```dart
// 使用代码生成
@riverpod
class User extends _$User {
  @override
  Future<UserModel?> build(String userId) async {
    final repo = ref.watch(userRepositoryProvider);
    return await repo.getUser(userId);
  }

  Future<void> update(UserModel user) async {
    state = AsyncValue.loading();
    state = await AsyncValue.guard(() async {
      final repo = ref.read(userRepositoryProvider);
      return await repo.updateUser(user);
    });
  }
}

// 使用 Provider
class UserPage extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final userAsync = ref.watch(userProvider('123'));

    return Scaffold(
      body: userAsync.when(
        data: (user) => Text(user?.name ?? ''),
        loading: () => CircularProgressIndicator(),
        error: (e, s) => Text('Error: $e'),
      ),
    );
  }
}
```

**对比其他方案**

| 方案 | 优点 | 缺点 | 选择 |
|------|------|------|------|
| **Riverpod** | 类型安全、无 BuildContext | 学习曲线 | ✅ 推荐 |
| **Provider** | 简单易用 | 无代码生成 | ⚠️ 备选 |
| **BLoC** | 结构清晰 | 样板代码多 | ❌ 不选 |
| **GetX** | 简单 | 类型不安全 | ❌ 不选 |

## 4. 网络层

### 4.1 Dio

```yaml
dependencies:
  dio: ^5.4.0
  pretty_dio_logger: ^1.3.0
  connectivity_plus: ^5.0.0
  cookie_jar: ^4.0.0
```

**选择理由**

```
✅ 功能强大: 拦截器、请求取消、文件上传
✅ API 友好: 链式调用，易于使用
✅ 良好支持: 活跃维护，文档完善
✅ 扩展性强: 支持自定义拦截器
```

**使用示例**

```dart
final dio = Dio(BaseOptions(
  baseUrl: 'https://api.example.com',
  connectTimeout: Duration(seconds: 30),
  receiveTimeout: Duration(seconds: 30),
));

// 添加拦截器
dio.interceptors.addAll([
  AuthInterceptor(),
  LogInterceptor(),
  ErrorInterceptor(),
]);

// GET 请求
final response = await dio.get('/users/123');

// POST 请求
final response = await dio.post(
  '/users',
  data: {'name': 'John'},
);
```

## 5. 数据存储

### 5.1 SQLite - sqflite

```yaml
dependencies:
  sqflite: ^2.3.0
  path_provider: ^2.1.0
  path: ^1.8.0
```

**选择理由**

```
✅ 关系型数据库: 适合结构化数据
✅ SQL 支持: 复杂查询
✅ 事务支持: 保证数据一致性
✅ 跨平台: 支持 iOS、Android
```

**使用示例**

```dart
final db = await openDatabase(
  'my_app.db',
  version: 1,
  onCreate: (db, version) async {
    await db.execute('''
      CREATE TABLE users (
        id TEXT PRIMARY KEY,
        name TEXT,
        email TEXT
      )
    ''');
  },
);

// 查询
final users = await db.query('users');

// 插入
await db.insert('users', {
  'id': '123',
  'name': 'John',
  'email': 'john@example.com',
});
```

## 9. 技术选型决策矩阵

### 9.1 决策因素

```
功能完整性: 30%  - 是否满足功能需求
性能:        25%  - 性能是否达标
易用性:      20%  - 开发体验如何
```
