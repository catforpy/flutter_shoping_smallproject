# 笔记编辑器 - 数据流程说明

## 📊 完整数据流程

```
用户选择图片/视频
    ↓
打印本地路径: /storage/emulated/0/.../image.jpg
    ↓
模拟上传到 OSS (打印日志)
    ↓
获得 URL: http://oss.example.com/images/abc123.jpg
    ↓
插入到 Quill 编辑器
    ↓
用户点击保存
    ↓
提取 Delta JSON
    ↓
打印 SQL 示例
    ↓
返回 JSON 字符串给上一级页面
```

## 💾 数据库存储

### 表结构

```sql
CREATE TABLE notes (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  title VARCHAR(200),
  content TEXT,           -- 存储 Delta JSON
  is_public BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  INDEX idx_user_id (user_id),
  INDEX idx_is_public (is_public)
);
```

### 实际存储示例

```json
// content 字段存储的内容
{
  "ops": [
    {"insert": "今天学习了 Flutter 富文本编辑器\n"},
    {"insert": {"image": "http://oss.example.com/image/file_1740647130-123456.jpg"}},
    {"insert": "\n"},
    {"insert": "重点内容：\n"},
    {"insert": {"video": "http://oss.example.com/video/file_1740647140-789012.mp4"}},
    {"insert": "\n"}
  ]
}
```

**关键点**：
- ✅ JSON 中只存储 URL 字符串
- ✅ 不存储实际文件内容
- ✅ 图片/视频在 OSS 中
- ✅ 数据库只存储文本 JSON，体积小

## 🔄 前端显示流程

### 从数据库读取并显示

```dart
// 1. 从数据库获取笔记内容
final note = await database.getNote(1);
final contentJson = note.content; // Delta JSON 字符串

// 2. 解析 JSON
final delta = Delta.fromJson(jsonDecode(contentJson));

// 3. 创建 QuillController
final controller = QuillController(
  document: Document.fromJson(delta.toJson()),
  selection: const TextSelection.collapsed(offset: 0),
);

// 4. 显示在 QuillEditor 中
QuillEditor(
  controller: controller,
  readOnly: true,  // 只读模式
)
```

## 🎯 实际对接后端时的改动

### 1. 修改上传服务

在 `file_upload_service.dart` 中：

```dart
// 替换模拟上传为真实上传
Future<UploadResult?> uploadFile(File file, {
  required String fileType,
  void Function(double progress)? onProgress,
}) async {
  // 1. 从后端获取上传凭证（如果需要）
  final uploadToken = await _getUploadToken();

  // 2. 上传到真实的 OSS
  final result = await _realUploadToOSS(
    file,
    token: uploadToken,
    onProgress: onProgress,
  );

  // 3. 返回真实的 URL
  return UploadResult(
    url: result.url,  // 真实的 CDN URL
    fileName: result.fileName,
    fileSize: result.fileSize,
    fileType: fileType,
  );
}
```

### 2. 修改保存逻辑

```dart
Future<void> _handlePublish() async {
  final deltaJson = _controller.document.toDelta().toJson();
  final jsonString = jsonEncode(deltaJson);

  // 发送到后端 API
  final response = await http.post(
    Uri.parse('https://api.example.com/notes'),
    body: {
      'content': jsonString,
      'is_public': _isPublic,
    },
  );

  if (response.statusCode == 200) {
    // 保存成功
    Navigator.pop(context);
  }
}
```

## 📌 注意事项

1. **当前 URL 是模拟的**：`http://oss.example.com/...`
2. **需要替换为真实 CDN 域名**：如 `https://cdn.yourdomain.com/...`
3. **本地路径只用于调试**：实际保存的是 URL
4. **JSON 格式是 Quill Delta**：前端和后端都要使用这个格式
